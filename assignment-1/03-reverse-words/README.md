# 3. Reverse the word order

Return a **new `String`** whose whitespace-separated words appear in reverse order. Collapse runs of whitespace to a single space; trim leading/trailing whitespace from the output.

```rust
pub fn reverse_words(sentence: &str) -> String;
```

### Examples

- `"hello world rust"` → `"rust world hello"`
- `"   one   two  "` → `"two one"`
- `""` → `""`
